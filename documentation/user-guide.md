## Getting Started

### Opening the Report

1. Navigate to the `reports` folder
2. Double-click on `library-report.pbix`
3. Power BI Desktop will launch automatically

## Navigating the Dashboard

The **Home Page** serves as your navigation hub with three main options:

- **Current Loans**: View all active loans
- **Students Review**: Analyze student borrowing patterns
- **Report Filters**: Access advanced filtering options

## Understanding Each Page

### Current Loans Page

Monitor all active book loans and identify overdue items.

**Key Sections:**
- Total Loans card
- Total Overdue Loans card
- Days Overdue slider
- Detailed loan table

### Report Filters Page

Apply advanced filtering across multiple dimensions.

**Filters Available:**
- School Year and Month
- Student Names

### Student Review Page

Analyze student borrowing patterns and trends over time.

**Features:**
- Total loans by student visualization
- Year-over-year comparison charts
- Detailed student loan history

## Using Filters

### Applying Filters

1. Navigate to Report Filters page
2. Select desired year and month
3. Choose specific students
4. View filtered results on other pages

### Clearing Filters

- Click "Select all" to reset
- Refresh the page

## Common Tasks

### Task 1: Find All Overdue Books

1. Go to Current Loans page
2. Check the "True" checkbox under "is Overdue"
3. Review the loan table

### Task 2: Check a Specific Student's Loans

1. Go to Report Filters page
2. Select the student name
3. Navigate to Current Loans or Student Review

### Task 3: Export Data

1. Click on any visual
2. Click the three dots (...)
3. Select "Export data"
4. Choose format and export

## Tips & Best Practices

- Start with broad view, then narrow with filters
- Use cross-filtering by clicking on visuals
- Refresh data regularly
- Clear filters when done analyzing
